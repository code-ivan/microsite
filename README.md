#  React app with React Router + Express.js


## Development

Install dependencies:

```sh
npm install
```

Start the project at [`http://localhost:3000`](http://localhost:3000).

```sh
npm start:client
```
Start the with server side project at [`http://localhost:9000`](http://localhost:9000).

```sh
npm start
```

## Testing

```sh
npm test
```
